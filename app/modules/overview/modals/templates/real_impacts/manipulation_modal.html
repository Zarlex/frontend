Manipulation

Cyberwar-Attacken können Funktionen in den Zielsystemen so verändern, dass sie nicht ihren ursprünglich programmierten Zweck, sondern den Zielen des Angreifers dienen.

Informationsmedien

Durch Veränderung von Informationen, bzw. durch das Ausfiltern bestimmter Informationsquellen kann die Öffentliche Meinung beeinflusst werden.

Wirtschaft

Die Wirtschaftsdaten von Unternehmen können so verändert werden, dass sie fehlgesteuert werden. Noch problematischer ist die Beeinflussung von Börsen- und Finanzdaten. Damit können eigene Spekulationserfolge erzielt werden, oder ganze Staatswirtschaften ruiniert werden.

Kritische Infrastruktur

Die Steuerung von Stromnetzen kann so manipuliert werden, dass einzelne regionale Wirtschaftsbereiche lahmgelegt werden können.
