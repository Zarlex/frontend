Sabotage

Von Sabotage spricht man dann, wenn die Manipulation der IT-Zielsystem zur Zerstörung dieser Systeme, oder zur Zerstörung der davon abhängigen Realsystem führt.

Informationsmedien

Das Blockieren von Internetportalen ist die einfachste Sabotageform. Sie verhindert den geplanten Informationsfluss.

Wirtschaft

Hat ein Angreifer die Kontrolle über die Steuerung eines Wirtschaftsunternehmens übernommen, kann er einzelne Produktionsstraßen und Maschinen so fehlsteuern, dass sie sich selbst zerstören.

Kritische Infrastruktur

Die Steuerung von Energienetzen kann so manipuliert werden, dass eine künstlich erzeugte Überlast zur Netzzerstörung führen kann. Kraftwerke können durch Fehlsteuerungen ebenfalls zerstört werden. Dies kann insbesondere bei Atomkraftwerken zu unüberschaubaren Gefahren führen.
